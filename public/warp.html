Warp Pipeline

1
Capture
Fetch data, store snapshot blob, compute content hash (SHA-256)
2
Extract
Apply rules/CSS/XPath/LLM hybrid to produce loose dictionary
3
Normalize
Convert currency, units, locale, whitespace, and number coercions
4
Map
Assign domain schema and perform field mapping transformations
5
Validate
JSON Schema + semantic rule engine validation with detailed results
6
Dedupe
Fingerprinting and clustering across envelope + payload data
7
Persist
Write to object store and index metadata to database
